  1. How do I cite CiteSeerExtractor?
    You can cite this paper: Kyle Williams, Lichi Li, Madian Khabsa, Jian Wu, Patrick C. Shih, C. Lee Giles. 2014. A Web Service for Scholarly Big Data Information Extraction. In 21st IEEE International Conference on Web Services (to appear).
  2. I've made some improvements to CiteSeerExtractor and think other people may find them useful. How do I get them included in CiteSeerExtractor?
    Great! Please fork CiteSeerExtractor on GitHub and make a pull request to have your changes incorporated into the codebase. We'll review them and let you what the outcome is.
  3. CiteSeerExtractor has been working for a while today and then it suddenly stopped. What's wrong?
    It's possible that the service has gone down; however, the fact that you're seeing the page suggests that it's still up. A more likely reason is that you're reached your maximum number of requests for the day. You can either wait and try again tomorrow or you could run your own CiteSeerExtractor service.
