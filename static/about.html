About

Extracting information from scholarly documents is an important and common task in academic digital libraries and publication analysis. For instance, citations need to be extracted for citation analysis and to build citation and co-authorship graphs. Similarly, header information, such as title, author, and abstracts are extracted for display and easy search. CiteSeerExtractor provides a uniform and straight-forward method for performing this information extraction via a standard RESTful interface.

CiteSeerExtractor runs as a standalone Web server (see the documentation for running details) and can either be included in any scientific document workflow.

CiteSeerExtractor is developed at the Pennsylvania State University

Tools

CiteSeerExtractor incorporates a series of open source tools, most of which are used in the CiteSeerX digital library. These tools are combined in CiteSeerExtractor in order to provide a simple workflow for information extraction.

The tools currently in use in CiteSeerExtractor are:

PDFBoxA Java library maintained by the Apache Foundation for extracting content from PDFs. In CiteSeerExtractor PDFBox is used for PDF to text conversion.http://pdfbox.apache.org/Apache License v2.0
