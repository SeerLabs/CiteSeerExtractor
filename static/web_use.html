Web Use

CiteSeerExtractor is available via the Web for testing and experimentation. It probably isn't very useful in production, but could be useful if you want to test it out or only extract information from a few documents. For production, we recommend that you use the RESTful API or (even better) run your own service.

The URL for extraction is available at http://citeseerextractor.ist.psu.edu/extractor.

From the URL you are able to upload a document and submit it. An XML document is returned with URLs for retrieving different parts of the document (header, citations, etc). See the description of the RESTful API for details of the types of resources that can be retrieved via the Web.
