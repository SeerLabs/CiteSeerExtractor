RESTful API Use

The CiteSeerExtractor RESTful API supports POST and GET methods for submitting documents and extracting information from them as well as a DELETE method for deleting a document. The base URL is http://citeseerextractor.ist.psu.edu/extractor.

REST Methods

The methods supported by CiteSeerExtractor are summarized in the Table below.

MethodURLDescriptionReturnsOptions
POST/Uploads a new PDF document either via a form or via bytestreamXML document with URIs to resource, resource_idmyfile=@filename (required for form POST)
GET/{resource_id}/pdfUsed to download original PDF of resourcePDF document for resourceN/A
GET/{resource_id}/headerExtracts the header information (authors, title, etc) from the resourceRepresentation of header informationoutput={xml (default) | json}
GET/{resource_id}/citationsExtracts the citations from the resourceRepresentation of the citationsoutput={xml (default) | json}
GET/{resource_id}/bodyExtracts the body (text excluding header and citations) from the resourceRepresentation of the bodyoutput={xml (default) | json}
GET/{resource_id}/textExtracts the full text from the resourceFull text of resourceN/A
DELETE/{resource_id}Deletes the resourceConfirmationN/A

Example

Here's a brief example of how the API can be used with cURL.

Submit Document
Via form: curl -F myfile=@/path/to/file.pdf "http://citeseerextractor.ist.psu.edu/extractor" Bytestream: curl -X POST --data-binary @/path/to/file.pdf "http://citeseerextractor.ist.psu.edu/extractor" Returns: <CSXAPIMetadata>
<pdf>http://citeseerextractor.ist.psu.edu/extractor/{resource_id}/pdf</pdf>
<header>http://citeseerextractor.ist.psu.edu/extractor/{resource_id}/header</header>
<citations>http://citeseerextractor.ist.psu.edu/extractor/{resource_id}/citations</citations>
<body>http://citeseerextractor.ist.psu.edu/extractor/{resource_id}/body</body>
<text>http://citeseerextractor.ist.psu.edu/extractor/{resource_id}/text</text>
</CSXAPIMetadata>

Retrieve Content
curl http://citeseerextractor.ist.psu.edu/extractor/{resource_id}/header (replace header with citations, body, etc) Returns: <?xml version="1.0" encoding="UTF-8"?>
<CSXAPIMetadata>
<algorithm name="SVM HeaderParse" version="0.2">
<title>Paper Title</title>
<authors>
<author>
<name>Author 1</name>
<affiliation>Author's affiliation</affiliation>
<address>Authors address</address>
...
...
...
</CSXAPIMetadata>

Delete
From time to time the uploaded documents are manually removed. However, you can explicitly remove a document if you want. curl -X DELETE http://citeseerextractor.ist.psu.edu/extractor/{resource_id} Returns:DELETED resource_id
