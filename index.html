Welcome to CiteSeerExtractor

CiteSeerExtractor is an automatic metadata and citations extractor for scholarly documents. It is currently in use in CiteSeerX, SeerSuite, and a number of other scholar digital library projects.

CiteSeerExtractor leverages existing extraction tools, such as SVMHeaderParse and ParsCit and can either be run from the command line, as a stand-alone Web server, or integrated into an existing application.

Citeseerextractor maintained by SeerLabs
